# 🧮 Calculator App - React Native

A simple calculator mobile application built with **React Native** and **Expo**.

This project allows users to perform basic mathematical operations with a clean and modern UI.

## 📱 Features

- ✅ Display screen for calculations
- ✅ Add numbers dynamically
- ✅ Basic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
- ✅ Calculate result
- ✅ Clear input
- ✅ Prevent division by zero
- ✅ Responsive calculator layout

## 🛠️ Technologies

- React Native
- Expo
- TypeScript
- Expo Router
